Help

Usage: type Help

you can also type: ?

Type help to bring up this screen. Type back to leave this screen.

Nearly all commands in this game consist of a verb and a noun component, and each help page shows an example. Also, each command can be called using one of its synonyms, which are listed on their respective help pages.

Move

Usage: Move Somewhere

you can also type: Go, Walk, Run

Move around the game world.

Remember: an active lifestyle is the cornerstone of health.

Load

Usage: Load a Story

Load a JSON story file. Just type the name of the file, and leave off the extension (the command takes care of that for you.)

Look

Usage: Look at Something

Look at something in the area or in your inventory. Use whenever you need a bit more information on something.
Failing that, it might at least be an interesting read.
Or not.

Use

Usage: Use Object or Use First Object with Second Object

Use an object in the area or in your inventory. You can also use objects in combination with each other. Some things like levers or doors are accessible with alternate commands (like pull and open.)

Take

Usage: Take and Object

you can also type: Get

Take an object and store it in your inventory for later use. Don't ask me why you can carry so much and I won't ask you where you put it all. If you find something and you don't know what to do with it, try using the Look command on it.
