How's about I tell you a story?

This is the first release!

In its current state, this engine only supports basic exploration-type text adventures along with simple object use.

Upcoming Features:

  • Taking objects & player inventory
  • NPCs and dialog
  • The ability to combine/use items together
  • Player input history
  • Saving and loading game progress

Load the example story by inputting "load story" or, if you've written your own story, you can load it by typing the name of the file after the load command (make sure it doesn't have any spaces in the name.)

If you need some help on how to play, just type "help" and press enter.

For more information on Alexandria, visit the project website.
