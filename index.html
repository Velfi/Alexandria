A browser based text-adventure engine.

Alexandria is a text adventure engine the runs in a web browser. It's coded in JavaScript and jQuery and it reads from story files coded in JSON. This project is inspired by all the text adventures and MUDs I played as a kid.

Currently, Alexandria is a one-woman project that I build in my spare time. Although I am currently the sole author of Alexandria, big thanks go to my partner Ashley, who is always there to support me.

The game uses a parser that accepts simple noun-verb phrases and turns them into commands. For the example:

>> take torch
You take the torch and place it in your pack.
>> examine torch
It is a simple thing comprised of an oil-soaked rag wrapped around a stick.

You can learn about each command by reading the in-game help screen (accessible at time entering the 'help' command.)

It's simple to write your own adventures. To get started, check out the guide or start reading the full documentation. I hope to make the JSON very simple so that anyone can write their own. Currently, the engine is capable of supporting simple text adventures, but I am adding new things all the time. I hope to release a first version soon. I'll get to work writing documentation soon, including 'How to set up a test environment' and 'How to write a story.JSON file.'
